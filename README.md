# Tic Tac Toe Game

A modern, interactive Tic Tac Toe game built with HTML, CSS, and JavaScript. Play against a friend in this classic game with a clean, responsive interface.

## Features

- Play against a friend in local multiplayer
- Player names customization
- Color-coded X's and O's for better visibility
- Responsive design that works on all devices
- Win detection and draw conditions
- Option to start a new game or reset the current one

## How to Play

1. Enter player names and click "Start Game"
2. Player 1 (X) goes first, followed by Player 2 (O)
3. Click on any empty cell to place your mark
4. The first to get 3 in a row (horizontally, vertically, or diagonally) wins!
5. If all cells are filled without a winner, the game ends in a draw
6. Use "New Game" to play again or "Reset Game" to start over

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
